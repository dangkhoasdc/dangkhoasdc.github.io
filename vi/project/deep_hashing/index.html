Hashing in content-based image retrieval

  1. Image hasing: a joint optimization for image embedding and hashing (CVPR17 paper).
  2. Deep Image Retrieval: a scheme for unified deep learning and image retrieval (ACM MM17 paper).
  3. Mobile version of Triangulation Embedding.
  4. Deep Hashing Image (In progress).
