C++20 Features

Range-based for statements with initializer

With C++20, we can initialize local variables inside the range-based for. For example:

{
    T thing = f();
    for (auto& x: thing.items()) {
        mutate(&x);
        log(x);
    }
}

With the new feature, we can shorten the code:

for (T thing= f(); auto& x : thing.items()) {
    mutate(&x);
    log(x);
}
