2017-09-13 notes, programming languages, programming

There are some notes about cool stuff in C++, and also some crazy things.

Object Oriented Programming

/* (1) */
T object(arg);

/* (2) */
T object{arg};

(1) can be used by all types of class and non-class. In contrast, (2) is only used to initialize non-class type.
